# Pokedex App - React Native

Se realizo una aplicacion de pokedex utilizando react native y expo la cual consume la api de https://pokeapi.co/api/v2/ en la cual se desgolosa el listado de todos lo pokemon una barra de busqueda filtros por tipo asi como la vista de detalles de los mismos y la seccion de favoritos.

## Instalacion (Contar con dispositivo android y habilitar opciones de desarrollador y depuracion por USB)
1. clonar el repositorio de git
2. entrar a la carpeta de la applicacion
3. realizar un yarn install 
4. realizar un npx expo run:android (para utilizar un dispositivo fisico)
5. realizar pruebas de funcionalidad
